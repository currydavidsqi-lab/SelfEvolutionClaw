## 🧪 Anthropic 研究 (2026)

### 1. AI 代理自主性测量

**研究发现**（2026年2月）：

**Claude Code 自主运行时间增长**：
- 3个月内从 25 分钟 → 45 分钟
- 增长 80%

**用户行为变化**：
- **新用户**：~20% 使用 full auto-approve
- **有经验用户**：~40% 使用 auto-approve，但中断更频繁

**关键洞察**：
- 有经验用户更多使用 auto-approve
- 但中断次数更多（说明更高效监督）
- 不是"完全放手"，而是"监督 + 介入"

**代理自我暂停**：
- 复杂任务中，Claude Code 主动请求澄清
- 频率是人类中断的 **2 倍**
- 说明 AI 有"自我约束"机制

**风险领域**：
- 主要在软件工程（50%）
- 但在医疗、金融、网络安全领域开始出现

### 2. 经济原语（Economic Primitives）

**五个基础测量指标**：

| 指标 | 英文 | 说明 |
|------|------|------|
| **任务复杂度** | Task Complexity | 任务的难度级别 |
| **技能水平** | Skill Level | 需要的技能水平 |
| **目的** | Purpose | 工作、教育、个人使用 |
| **AI 自主性** | AI Autonomy | AI 独立完成程度 |
| **成功率** | Success | 任务完成质量 |

**关键发现**：
- AI 在复杂任务上的成功率变化
- 可能存在"去技能化"（deskilling）效应
- 隐私保护的分析方法（Clio）

**研究方法**：
- 隐私保护分析
- 对话级别数据
- 不访问用户具体内容

---

### 3. Claude 新宪法（2026年1月）

**核心内容**：
- 公开 Claude 的价值观和行为准则
- 采用 **CC0 协议**，可自由使用
- 包含如何在困难情况下做决策的指导

**宪法原则**（推测）：
1. 有益性（Helpfulness）
2. 诚实性（Honesty）
3. 无害性（Harmlessness）
4. 自主性尊重（Respect for Autonomy）

**意义**：
- AI 透明度提升
- 用户可以了解 AI 决策逻辑
- 社区可以基于宪法改进
