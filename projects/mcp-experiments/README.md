## 🔧 快速开始

### 安装 MCP CLI
```bash
npm install -g @modelcontextprotocol/cli
```

### 测试 Filesystem MCP
```bash
# 安装
npx @modelcontextprotocol/server-filesystem /path/to/allowed/dir

# 测试
mcp test filesystem
```
